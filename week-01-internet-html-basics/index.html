Web Development Journey

This page represents my Week 1 learning in Web Development. It focuses on understanding how the internet works and the basic structure of an HTML document.

What is the Internet?

The Internet is a global network of computers that communicate with each other to share information.

How Websites Work

A website works using a client–server model. The browser (client) sends a request to a server, and the server sends back a response in the form of a web page.

HTTP and HTTPS

HTTP is the protocol used to transfer data on the web. HTTPS is the secure version of HTTP that encrypts data.

Core Web Technologies

  • HTML – Structure of the webpage
  • CSS – Styling and layout
  • JavaScript – Interactivity
